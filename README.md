# Paper Trail
A warioware style microgame that's paper themed!

<img width="297" height="230" alt="image" src="https://github.com/user-attachments/assets/541fbd32-c22d-4906-91ef-bbb32c9630ae" />

### How to play:
You have 3 lives. your goal is to complete 2 minigames before the time runs out.
Requires keyboard and mouse.
